Weather App

This application was crafted by using HTML, CSS, JavaScript, Bootstrap and API integration. The app provides users with a comprehensive view of the current weather conditions, displaying the current temperature along with the minimum and maximum temperatures for the day. It also offer information such as the precise timings for sunrise and sunset, wind speed, and humidity levels. Additionally, the app looks to the future with a 5-day forecast feature, furnishing users with anticipated minimum and maximum temperatures for the upcoming days.

World Clock App

Maria's world clock app is a versatile and user-friendly tool that displays the hour, minutes, and seconds for multiple cities across different time zones, such as Los Angeles, Milan and Sydney. Users can personalize their experience by adding or removing cities from the list.
